Tldraw.Blazor v1.0

Infinite canvas whiteboard SDK for Blazor, ported from tldraw — powered by SkiaSharp. Pure C# with minimal JS interop.

🎨 8 shape types
🛠️ 8 tools
🔗 Arrow bindings
↩️ Undo/Redo
📋 Clipboard
📱 Touch support
🌙 Dark mode
💾 Auto-save

Installation #

Prerequisites

  • .NET 10 SDK
  • Visual Studio 2022 or VS Code with C# extension

Clone & Build

bash 
git clone https://github.com/your-repo/Tldraw.Blazor.git
cd Tldraw.Blazor
dotnet build

Run

bash 
dotnet run --project src/Tldraw.Blazor.App --urls "http://localhost:5000"

Open http://localhost:5000 in your browser.

Quick Start #

Use as a Component

razor 
@using Tldraw.Blazor.Skia
@using Tldraw.Blazor.Core.Editor
@using Tldraw.Blazor.Core.Store

<TldrawCanvas Editor="_editor" />

@code {
    private Editor _editor = new();

    protected override void OnInitialized()
    {
        // Create shapes programmatically
        _editor.CreateShape("geo", 100, 100);
        var rect = _editor.Store.GetAllShapes().Last();
        rect.Width = 200;
        rect.Height = 120;
        rect.Style.Fill = "#E3F2FD";
        rect.Style.Color = "#1565C0";
        rect.Style.BorderRadius = 8;
        if (rect.Props is TLGeoProps props)
        {
            props.GeoType = "rectangle";
            props.Text = "Hello!";
        }
    }
}
💡 Full demo app See src/Tldraw.Blazor.App/Pages/Home.razor for a complete example with toolbar, style panel, context menu, and all features.

Editor #

The Editor class is the central engine. It owns the store, camera, tools, selection, and history.

csharp
var editor = new Editor();

// Create shapes
editor.CreateShape("geo", 100, 100);

// Switch tools
editor.SetActiveTool("draw");

// Undo/Redo
editor.Undo();
editor.Redo();

// Clipboard
editor.Copy();
editor.Paste();

Store & Records #

The TLStore is a reactive record store. All shapes, pages, bindings, and assets are TLRecord subclasses.

csharp
// Get all shapes
var shapes = editor.Store.GetAllShapes();

// Listen for changes
editor.Store.Changed += changes => {
    foreach (var change in changes)
        Console.WriteLine($"{change.Type}: {change.RecordId}");
};

// Snapshot (for undo/save)
var snapshot = editor.Store.GetSnapshot();
editor.Store.LoadSnapshot(snapshot);

Shapes #

8 built-in shape types, each with a ShapeUtil that handles rendering, bounds, and hit-testing.

Shape TypeUtil ClassDescription
"geo"GeoShapeUtilRectangle, ellipse, diamond, star, hexagon, triangle
"draw"DrawShapeUtilFreehand strokes with smooth bezier curves
"text"TextShapeUtilMulti-line text with alignment
"note"NoteShapeUtilSticky note with shadow and fold corner
"frame"FrameShapeUtilFrame container with label
"line"LineShapeUtilPolyline (straight segments)
"arrow"ArrowShapeUtilArrow with arrowhead and bindings
"image"ImageShapeUtilImage from data URI with bitmap cache

Creating Shapes

csharp
// Simple creation
editor.CreateShape("geo", 100, 100);

// With custom properties
var shape = editor.CreateShape("geo", 200, 200);
shape.Width = 300;
shape.Height = 200;
shape.Style.Fill = "#E3F2FD";
shape.Style.Color = "#1565C0";
shape.Style.BorderRadius = 12;
if (shape.Props is TLGeoProps geo)
{
    geo.GeoType = "ellipse";
    geo.Text = "Custom Shape";
}

Tools #

All tools use hierarchical StateNode state machines with transitions.

ToolKeyStatesDescription
SelectToolVIdle → Pointing → Dragging / Brush / Resizing / Rotating / ArrowEditingSelect, move, resize, rotate shapes
DrawToolDIdle → DrawingFreehand drawing
GeoToolGIdle → CreatingCreate geometric shapes by drag
HandToolHIdle → PanningCanvas panning
TextToolTIdle → EditingCreate and edit text
ArrowToolAIdle → DrawingCreate arrows with bindings
EraserToolEIdle → ErasingDelete shapes
💡 Space+Drag Holding Space and dragging pans the canvas from any tool — no need to switch to Hand tool.

Camera #

The Camera manages viewport position and zoom level.

csharp
// Pan
editor.Camera.Pan(dx, dy);

// Zoom at screen point
editor.Camera.ZoomAt(screenPoint, delta);

// Coordinate conversion
var world = editor.Camera.ScreenToWorld(screenX, screenY);
var screen = editor.Camera.WorldToScreen(worldX, worldY);

// Reset
editor.Camera.Reset();

Selection & Transform #

Selecting

  • Click — select single shape
  • Shift+Click — toggle shape in selection
  • Drag on empty — rubber-band selection
  • Ctrl+A — select all

Resizing

  • Drag any of 8 handles (corners + edge midpoints)
  • Shift — constrain proportions (corners only)
  • Alt — resize from center
  • Escape — cancel, restore original bounds

Rotating

  • Drag the rotation handle (circle above top-center)
  • Shift — snap to 15° increments

Moving

  • Drag selected shapes to move
  • All selected shapes move together
  • Arrow bindings auto-update when bound shapes move

Arrow System #

Creating Arrows

Press A and drag from one point to another. If endpoints are near shapes, they auto-bind.

Bindings

  • Green dot — endpoint is bound to a shape
  • Blue dot — endpoint is unbound
  • When a bound shape moves, the arrow endpoint follows

Editing Endpoints

  1. Switch to Select tool (V)
  2. Click on an arrow endpoint dot
  3. Drag to move the endpoint
  4. Release near a shape to rebind
  5. Press Escape to cancel

Undo / Redo #

Snapshot-based undo system with 50 steps maximum.

Styling #

Select a shape to see the Style Panel on the right. Change:

  • Stroke color — 8 color presets
  • Fill color — 8 presets + "no fill"
  • Stroke width — slider 0.5–10
  • Opacity — slider 0%–100%
  • Dash pattern — solid / dashed / dotted
  • Border radius — slider 0–50 (geo shapes)
  • Font size — slider 8–72 (text/note shapes)

Toolbar #

The top toolbar provides:

  • 12 tool buttons — Select, Hand, Draw, Rect, Ellipse, Diamond, Star, Hex, Tri, Text, Arrow, Eraser
  • Export/Import — JSON, PNG, Image
  • Zoom controls — +/−, Fit, 1:1
  • Shortcuts help — ⌨️ toggle
  • Dark mode — 🌙 toggle

Style Panel #

Appears on the right when a shape is selected. See Styling for details.

Context Menu #

Right-click on the canvas to access:

  • With selection: Copy, Cut, Paste, Delete, Z-order, Group/Ungroup, Alignment, Distribution
  • Without selection: Paste, Select All, Undo, Redo

Minimap #

Overview panel in the bottom-left corner showing all shapes and the current viewport indicator.

Dark Mode #

Click the 🌙 button in the toolbar to toggle dark mode. Affects toolbar, panels, context menu, minimap, and pages panel.

Persistence #

  • Auto-save: Every 5 seconds to localStorage
  • Auto-load: On page startup, restores from localStorage
  • Manual save: Click 💾 JSON button → copies to clipboard

Image Loading #

Click the 🖼️+ Image button to open a file picker. Selected image is:

  1. Read as base64 data URI via JS interop
  2. Dimensions extracted
  3. Stored as TLAssetRecord
  4. Rendered as SKBitmap with caching
