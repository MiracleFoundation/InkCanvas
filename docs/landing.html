@using Tldraw.Blazor.Core.Editor

<TldrawCanvas Editor="_editor" />

@code {
    private Editor _editor = new();

    protected override void OnInitialized()
    {
        // Create shapes programmatically
        _editor.CreateShape("geo", 100, 100);
        var rect = _editor.Store
            .GetAllShapes().Last();
        rect.Style.Fill = "#E3F2FD";
    }
}

Why Tldraw.Blazor

Everything you need for infinite canvas

Ported from the tldraw SDK to pure C# — a production-grade canvas engine for .NET developers.

01

Canvas infrastructure

Viewport culling, shape caching with SKBitmap, adaptive grid rendering. Handles thousands of shapes without frame drops.

02

Accelerate development

8 shape types, 8 interactive tools, arrow bindings, undo/redo, clipboard, and touch support — all built in. Skip months of canvas engineering.

03

Customize, extend, control

Reactive store with change events. Shape utils for custom rendering. Style system with color, fill, stroke, opacity, dash, radius, and font controls.

04

Pure C# — no JS required

Core logic runs entirely in C# via SkiaSharp. Hardware-accelerated rendering with minimal JavaScript interop. Build canvas apps the .NET way.
